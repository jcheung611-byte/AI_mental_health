# AI Voice Assistant - Project Root

This folder contains the AI Voice Assistant project - a warm, safe voice AI with emotional intelligence.

## Structure

- `frontend/` - Next.js application (the main app)
- `ChatGPT initial brainstorming.md` - Original project vision and planning

## Quick Start

```bash
cd frontend
npm install
# Set up .env.local with your Portkey API key
npm run dev
```

See `frontend/README.md` for full documentation.
